# Product Overview

A Zabbix-like network monitoring system designed for college campus network infrastructure.

## Purpose
Monitor network switches via SSH to track performance metrics, detect issues, and generate alerts.

## Key Features
- SSH-based monitoring of network switches
- Real-time metrics: CPU usage, memory usage, uptime, interface statistics
- Web dashboard for visualization
- Automated alert system with configurable thresholds
- Historical data tracking
- Support for multiple switch vendors (Cisco, Arista, Juniper)

## Target Users
Network administrators managing college campus switches and network infrastructure.
